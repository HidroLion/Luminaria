Primer prototipo funcional del juego desarrollado en la Plataforma
Unity en su Version 2019.4.6f1.

Lenguaje Utilizado: C#

Recursos Externos Utilizados:
	Paquete de Recursos "Cinamachine" de Unity
	Paquetes de Recursos Básicos de Unity2D

Programas Utilizados:
	Unity (Motor de Desarrollo)
	Visual Studio 2019 (Scripting)
	Krita (Arte)
	GitHub Desktop (Repositorio)
	Microsoft Word (Documentación)
	Dia (Diagramas)

El primer prototipo posee Bugs que se deben arreglar en el futuro.

[POO - Arquitectura]

El proyecto se centra en POO en el principio de concordancia.
"Multiples Scripts cortos para funciones muy especificas"

La comunicación entre Scripts respeta los principios de optimización
básicos. NO se utilizaron patrones de programación.
